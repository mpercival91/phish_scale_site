About
The Phish Scale is the result of Research conducted by NIST researchers Michelle P. Steves, Kristen Greene and Mary Theofanos which aims to factor difficulty into the analysis of phishing email detectability. The below calculator was built by Michael Percival as a proof of concept to put this research into practice. The intention of this site to provide others with the means to use the Phish Scale and further refine the Phish Scale concept. This site runs purely in HTML, CSS and JS with no server side code therefore any uploaded images are not stored.

Observed Cues
 Spelling and grammar irregularities
Does the message contain spelling or grammar errors, including mismatched lurality?
  Inconsistency
Are there inconsistencies contained in the email message?
  Attachment type
Is there a potentially dangerous attachment?
  Sender display name and email address
Does a display name hide the real sender?
  URL hyperlinking
Is there text that hides the true URL in a hyperlink?
  Domain spoofing
Is a domain name used in addresses or links plausibly similar to a legitimate entity’s domain?
  No/minimal branding and logos
Is appropriate branding missing?
  Logo imitation or out-ofdate branding/logos
Do any branding elements appear to be an imitation or out-ofdate?
  Unprofessional looking design or formatting
Does the design and formatting violate any conventional professional practices?
  Legal language/copyright info/disclaimers
Are any inappropriate security indicators or icons present?
  Distracting detail
Does the message contain any detailed aspects that are not central to the content?
  Requests for sensitive information
Does the message contain a request for any sensitive information, including personally identifying information or credentials?
  Sense of urgency
Does the message contain time pressure, including implied?
  Threatening language
Does the message contain a threat, including an implied threat?
  Generic greeting
Does the message lack a greeting or lack personalization in the message?
  Lack of signer details
Does the message lack detail about the sender, such as contact information?
  Humanitarian appeals
Does the message make an appeal to help others?
  Too good to be true offers
Does the message offer anything that is too good to be true, such having won a contest, lottery, free vacation and so on?
  You’re special
Does the message offer anything just for you?
  Limited time offer
Does the message offer anything for a limited time?
  Mimics a work or business process
Does the message appear to be a work or business-related process?
  Poses as friend, colleague, authority figure
Does the message appear to be from a friend, colleague, boss or other authority entity?
